# Dynamic Programming Question List

## Features

- 最优子结构：问题的最优解包含其子问题的最优解。
- 重叠子问题：在求解过程中，会多次遇到相同的子问题，这些子问题的解可以被复用。
- 无后效性：一旦某个状态确定，后续决策只与当前状态有关，与如何到达该状态的路径无关。

- 典型问题: 计数问题, 最值问题, 存在性问题等等
