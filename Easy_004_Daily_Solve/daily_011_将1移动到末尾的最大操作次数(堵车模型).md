# 将 1 移动到末尾的最大操作次数

[3228. 将 1 移动到末尾的最大操作次数](https://leetcode.cn/problems/maximum-number-of-operations-to-move-ones-to-the-end/description/)

--- 题目描述 ---

给你一个 二进制字符串 s。

你可以对这个字符串执行 任意次 下述操作：

- 选择字符串中的任一下标 i（ i + 1 < s.length ），该下标满足 s[i] == '1' 且 s[i + 1] == '0'。
- 将字符 s[i] 向 右移 直到它到达字符串的末端或另一个 '1'。
  例如，对于 s = "010010"，如果我们选择 i = 1，结果字符串将会是s = "000110"。

返回你能执行的 最大 操作次数。

--- 测试用例 ---

```text
示例 1：
输入： s = "1001101"
输出： 4
解释：
可以执行以下操作：
选择下标 i = 0。结果字符串为 s = "0011101"。
选择下标 i = 4。结果字符串为 s = "0011011"。
选择下标 i = 3。结果字符串为 s = "0010111"。
选择下标 i = 2。结果字符串为 s = "0001111"。

示例 2：
输入： s = "00111"
输出： 0
```

--- 说明提示 ---

```text
1 <= s.length <= 10^5
s[i] 为 '0' 或 '1'。
```

--- 题目解答 ---

## 堵车模型

把 1 当作车，想象有一条长为 n 的道路上有一些车。

题意：把所有的车都开到最右边。例如 011010 最终要变成 000111。

- 如果从最右侧开始操作( 优先移动右侧能动的车辆 )
     011010
  -  011001
  -  010011
  -  000111
  - 总操作数3次

- 如果从最左侧开始移动 ( 优先移动左侧能动的车辆 )
  - 很显然左侧可移动车辆移动后,当前车辆的左侧所有车辆均可移动
     011010
  -  010110
  -  001110
  -  001101
  -  001011
  -  000111
  - 总操作数5次

- 算法流程:
  - 当s[i] == 1时: cnt += 1 统计当前位置左侧车辆数
  - 当s[i] == 0 and s[i-1] == 1时:左侧所有车辆均可操作一次 ans += cnt
  - 继续当前操作,直到s遍历结束

```python
class Solution:
    def maxOperations(self, s: str) -> int:

        ans = 0
        cnt = 0

        for i, v in enumerate( s ):
            if v == '1': cnt += 1
            else:
                if i > 0 and s[ i - 1 ] == '1': ans += cnt
        return ans
```
